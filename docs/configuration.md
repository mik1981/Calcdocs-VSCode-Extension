# Configuration

Key settings:

## Core

- `calcdocs.enabled`: global on/off.
- `calcdocs.scanInterval`: periodic analysis interval.
- `calcdocs.ignoredDirs`: directories excluded from workspace scan.

## Backend / Wrappers

- `calcdocs.useClangd`: enable clangd wrapper backend.
- `calcdocs.enableCppProviders`: enable C/C++ providers.

## Cache & Performance

- `calcdocs.cppCacheMaxEntries`: max LRU entries for preprocessed C/C++ cache.
- `calcdocs.resourceStatusMode`
- `calcdocs.resourceCpuThreshold`

## C/C++ UI

- `calcdocs.cpp.codeLens.*`
- `calcdocs.cpp.hover.*`
- `calcdocs.inline.ghost.enabled`

## Inline Calculations

- `calcdocs.inline.codeLens.*`
- `calcdocs.inline.hover.*`
- `calcdocs.inline.diagnostics.level`

## Formatting

- `calcdocs.thousandsSeparator`
