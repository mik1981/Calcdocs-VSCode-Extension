# 🚀 Getting Started

This guide helps you start using CalcDocs in under 2 minutes.

---

## 1. Install the extension

Install CalcDocs from the VS Code Marketplace or via `.vsix`.

---

## 2. Open your project

CalcDocs works automatically on:

- C/C++ files (`.c`, `.h`, `.cpp`, `.hpp`)
- YAML formula files (`formula*.yaml`, `formulas*.yml`)

---

## 3. Hover to see computed values

Hover over:

- `#define`
- `const` variables
- YAML formula symbols

👉 You will instantly see:

- expanded expression  
- substituted values  
- final numeric result (decimal + hex)  

---

## 4. Optional: trigger manual refresh

Open Command Palette:
`CalcDocs: Force Formula Refresh`

---

## 5. Try inline calculations

Write inside comments:

```c
// calc: RPM * 0.10472
```

👉 CalcDocs evaluates it automatically.

✅ That’s it

No configuration required.

CalcDocs starts analyzing your project immediately.
